# Welcome!

This Java template lets you get started quickly with a simple one-page playground.

```java runnable
// { autofold
public class Main {


public static void modif(int value){
   value = 2;
}

public static void main(String[] args) {
// }

int value = 0;
modif(value);
System.out.println(value);

//{ autofold
}

}
//}
```
